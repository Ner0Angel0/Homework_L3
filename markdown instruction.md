# Инструкция по работе с markdown

## Выделение текста
Чтобы выделить текст курсивом, необходимо обрамить его в звездочками(*). Например, *вот так*.

Чтобы выделить текст полужирным, нужно обрамить его уже двумя звездочками (**). **Вот так.п**
